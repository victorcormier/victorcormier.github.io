ADV Films aquired the rights to distribute the english-language version of Makoto Shinkai's 2007 film 5 Centimeters Per Second. I was asked to handle the design work for the required advertisements, DVD packaging, DVD menu, etc.

A fan of Shinkai's early work, this project was a pleasure to work on.
